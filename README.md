# roya-site

All HTML and CSS. SASS preprocessor for styles:
````
$ cd roya-site
$ sass --watch ./sass:./css
````

If viewing on a local machine, use a simple server, like [http-server](https://github.com/indexzero/http-server), so relative links work properly.
````
$ http-server -p 3000
// url: localhost:3000/
````
